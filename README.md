# Tienda de Frutas y Verduras

Este es un proyecto de React para una tienda en línea de frutas y verduras. Cumple con los siguientes requerimientos:
- 4 secciones principales con subsecciones.
- Productos con imágenes y descripciones.
- Carrito de compras con opción de compra por peso.
